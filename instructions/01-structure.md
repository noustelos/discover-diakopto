# Project Structure: Discover Diakopto
Create the following folder and file hierarchy in the root directory:

discoverdiakopto/
├── index.html, explore.html, things-to-do.html, beaches.html
├── eat-drink.html, stay.html, events.html, gallery.html
├── plan-your-visit.html, about.html, 404.html
├── assets/
│   ├── css/ (main.css, layout.css, components.css, responsive.css)
│   ├── js/ (main.js, navigation.js, gallery.js, animations.js)
│   ├── images/ (hero, experiences, beaches, gallery, villages)
│   └── icons/
├── components/
│   ├── header.html, footer.html, hero.html
│   ├── experience-card.html, gallery-grid.html
└── data/
    ├── events.json, gallery.json, locations.json
